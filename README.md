🖥️ Practical Utility Scripts

This repository also contains workflow automation scripts for educational and practical purposes.

Ubuntu Dual Access (RDP + VNC Optimized)

A GitHub Actions workflow that automatically provisions an Ubuntu desktop environment with:

- XFCE4 Desktop Environment
- XRDP support for Microsoft Remote Desktop
- TightVNC Server support
- Tailscale integration for secure remote access
- Mobile-data optimized configuration
- Temporary cloud-based Ubuntu workstation

This workflow is intended for:

- Remote desktop experiments
- Linux learning and practice
- Software testing
- Educational demonstrations
- Temporary development environments
